# Code of Conduct

## Purpose

`tovli` should be a useful, respectful, and low-friction place to collaborate on
local-first retrieval and RAG tooling.

## Expected Behavior

Contributors and maintainers are expected to:

- Treat others with respect.
- Focus criticism on ideas, code, behavior, and evidence.
- Assume good intent while still being precise about technical concerns.
- Keep discussions relevant to the project.
- Respect boundaries around private data, security reports, and unpublished
  work.
- Help maintain a practical, welcoming environment for new contributors.

## Unacceptable Behavior

The following behavior is not acceptable:

- Harassment, intimidation, or personal attacks.
- Insults related to identity, background, ability, or experience level.
- Publishing private communication without consent.
- Sharing another person's private information.
- Deliberately derailing technical discussions.
- Encouraging unsafe handling of secrets, private documents, or security
  reports.

## Enforcement

Maintainers may remove content, close discussions, reject contributions, or
limit participation when behavior harms the project or its contributors.

If you need to report a conduct concern, use the maintainer contact listed in
the repository profile. If no private contact is available yet, open a brief
public issue asking for a maintainer contact without including sensitive
details.

Reports will be handled as privately and promptly as the project setup allows.
